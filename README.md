# Password Strength Checker

A simple web application that analyzes the strength of a password using a Flask-based backend and a responsive frontend interface.  
The tool measures password entropy, classifies strength levels, and provides actionable suggestions for improvement.

---

## Features

- Password strength analysis based on entropy and character variety  
- Recommendations for creating stronger passwords  
- Real-time frontend interaction  
- Backend API built with Flask  
- Clean UI with a strength meter bar

---

## Tech Stack

**Frontend**
- HTML  
- CSS  
- JavaScript  

**Backend**
- Python  
- Flask  
- Flask-CORS
